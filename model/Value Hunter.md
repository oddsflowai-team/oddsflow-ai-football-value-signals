# Value Hunter Model (Calm & Steady)

## Overview
The Value Hunter model is OddsFlowâ€™s systematic and repeatable model style.

It focuses on consistently identifying small-to-medium pricing inefficiencies, emphasizing long-term repeatability rather than short-term volatility.

The Value Hunter model does **not** predict match scores.  
It estimates fair odds and identifies mispriced bookmaker odds using a calm, steady approach.

---

## Risk profile
- Moderate volatility
- Emphasis on consistency
- Designed for repeatable value detection
- Avoids extreme variance

This model aims to balance signal frequency and stability.

---

## Typical market fit
The Value Hunter model is commonly applied to:
- **1X2 (Moneyline)**
- **Asian Handicap**
- Selected **Over / Under** markets

Market fit can vary by league and season.

---

## How signals are published
Signals generated by the Value Hunter model include:
- `model_style = VALUE_HUNTER`
- Market odds (book odds)
- OddsFlow fair odds estimate
- Edge percentage
- Confidence indicator
- Timestamp (UTC, published before kickoff)

---

## Verification
Performance of the Value Hunter model can be verified by:
- Filtering logs where `model_style = VALUE_HUNTER`
- Reviewing long-term performance over sufficient samples
- Auditing results by market type and confidence tier

All detected signals are published transparently.

---

## Important notes
The Value Hunter model is designed for long-term analysis.

Consistent value detection does not eliminate risk, and individual outcomes may vary.
